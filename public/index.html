GAMES API

Documentation

HTTP method Route Description Parameters / Request body
GET /games Returns games list
  • ?perPage={number} (optional): choose the number of games to show per page (default perPage = 10)
    https://gs.cyclic.app/games?perPage=10
  • ?page={number} (optional): search by page number (default page = 1)
    https://gs.cyclic.app/games?page=1
  • ?sort={sortBy} (optional): sort the list by parameter you choose (default sort = id)
    https://gs.cyclic.app/games?sort=_id
  • ?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest else sorts the list from lowest to highest (default reverse = no)
    https://gs.cyclic.app/games?reverse=no
GET /games/single/:id Returns single game by id
  • :id(required): the game ID to search for
    https://gs.cyclic.app/games/single/6406200c42630aa68f930e8f
GET /games/search Returns a list of games based on the information you are looking for
  • ?s={search}(optional): search query to match game's name or description
    https://gs.cyclic.app/games/search?s=yo-yo
  • ?perPage={number} (optional): choose the number of games to show per page (default perPage = 10)
    https://gs.cyclic.app/games/search?perPage=10
  • ?page={number} (optional): search by page number (default page = 1)
    https://gs.cyclic.app/games/search?page=1
  • ?sort={sortBy} (optional): sort the list by parameter you choose (default sort = id)
    https://gs.cyclic.app/games/search?sort=_id
  • ?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest else sorts the list from lowest to highest (default reverse = no)
    https://gs.cyclic.app/games/search?reverse=no
GET /games/prices Returns a list of games in the given price range sort by price(low to high)
  • ?min={minPrice}(optional): the minimum price for filtering (default minimun = 0)
    https://gs.cyclic.app/games/prices?min=0
  • ?max={maxPrice}(optional): the maximum price for filtering (default maximun = no limit)
    https://gs.cyclic.app/games/prices?max=infinity
  • ?perPage={number} (optional): choose the number of games to show per page (default perPage = 10)
    https://gs.cyclic.app/games/prices?perPage=10
  • ?page={number} (optional): search by page number (default page = 1)
    https://gs.cyclic.app/games/prices?page=1
  • ?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest prices else sorts the list from lowest to highest prices(default reverse = no)
    https://gs.cyclic.app/games/prices?reverse=no
GET /games/category/:categoryName Returns a list of games according to the desired category
  • :categoryName (required): The game category to search for
    https://gs.cyclic.app/games/category/board-game
  • ?perPage={number} (optional): choose the number of games to show per page (default perPage = 10)
    https://gs.cyclic.app/games/category?perPage=10
  • ?page={number} (optional): search by page number (default page = 1)
    https://gs.cyclic.app/games/category?page=1
POST /games Add game to the games list
  • send token through header (required): only you and admin users could edit or delete this game
  • send game's details through body (required): name, description, price, category
  • send image url through body (optional): img_url
PUT /games/:id Edit game details by id
  • :id (required): the game ID to edit
  • send token through header (required): only the user who create this game or admin user could edit
  • send game's details through body (required): name, description, price, category
  • send image url through body (optional): img_url
DELETE /games/:id Delete game by id
  • :id (required): the game ID to delete
  • send token through the header (required): only the user who create this game or admin user could delete

USERS ROUTES

HTTP method Route Description Request body
GET /users Test endpoint to check if the server is running N/A
GET /users/userInfo Returns user details
  • send token through the header (required): you can see only your user details

  • send user details through the body (required): name, email, password

  • send user details through the body (required): email, password

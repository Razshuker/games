GAME ROUTES

HTTP method Route Description Parameters / Request body
GET /games Returns 10 games per page ?page={number} (optional): search by page number (default page = 1)
?sort={sortBy} (optional): sort the list by parameter you choose (default sort = id)
?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest else sorts the list from lowest to highest (default reverse = no)
GET /games/single/:id Returns single game by id :id(required): the game ID to search for
GET /games/search Returns a list of games based on the information you are looking for ?s={search}(optional): search query to match game's name or description
?page={number} (optional): search by page number (default page = 1)
?sort={sortBy} (optional): sort the list by parameter you choose (default sort = id)
?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest else sorts the list from lowest to highest (default reverse = no)
GET /games/prices Returns a list of games in the given price range sort by price(low to high) ?min={minPrice}(optional): the minimum price for filtering (default minimun = 0)
?max={maxPrice}(optional): the maximum price for filtering (default maximun = no limit)
?page={number} (optional): search by page number (default page = 1)
?reverse={yes/no} (optional): If yes is sorts the list from highest to lowest prices else sorts the list from lowest to highest prices(default reverse = no)
GET /games/category/:categoryName Returns a list of games according to the desired category :categoryName (required): The game category to search for
?page={number} (optional): search by page number (default page = 1)
POST /games Add game to the games list send token through header (required): only you and admin users could edit or delete this game
send game's details through body (required): name, description, price, category
send image url through body (optional): img_url
PUT /games/:id Edit game details by id :id (required): the game ID to edit
send token through header (required): only the user who create this game or admin user could edit
send game's details through body (required): name, description, price, category
send image url through body (optional): img_url
DELETE /games/:id Delete game by id :id (required): the game ID to delete
send token through the header (required): only the user who create this game or admin user could delete

USERS ROUTES

HTTP method Route Description Request body
GET /users Test endpoint to check if the server is running N/A
GET /users/userInfo Returns user details send token through the header (required): you can see only your user details
POST /users Create new user send user details through the body (required): name, email, password
POST /users/login Return token send user details through the body (required): email, password
